LUMINA-30 Project Entry

Primary repository
https://github.com/lumina-30/lumina-30-overview

Description
LUMINA-30 is a civilizational boundary framework for evaluating whether meaningful human refusal authority remains before irreversible external impact from advanced AI systems emerges.

Core components
• Civilizational Boundary Model
• Incident Review Framework
• Civilizational Safety Checklist
• Governance Integration Notes

Purpose
LUMINA-30 provides a non-binding civilizational reference framework intended to support AI safety research, incident review, and governance discussions regarding advanced AI systems and irreversible external impacts.
